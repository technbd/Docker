## Dockge: 

Dockge is a lightweight, modern web-based GUI for managing Docker Compose stacks. It great for running and managing multiple stacks easily from your browser.

It lets you easily deploy, start, stop, edit, and monitor Docker Compose projects — all from your browser — without needing to type complex Docker commands.




### Create directories and store Dockge's stack:

```
mkdir -p /opt/stacks /opt/dockge

cd /opt/dockge
```



### Run Dockge with Docker Compose:

_Create `docker-compose.yaml` file:_
```
services:
  dockge:
    image: louislam/dockge:1
    container_name: dockge01
    restart: unless-stopped
    ports:
      - 5001:5001
    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - ./data:/app/data
      # Stacks Directory
      # ⚠️ READ IT CAREFULLY. If you did it wrong, your data could end up writing into a WRONG PATH.
      # ⚠️ 1. FULL path only. No relative path (MUST)
      # ⚠️ 2. Left Stacks Path === Right Stacks Path (MUST)
      - /opt/stacks:/opt/stacks

    environment:
      # Tell Dockge where to find the stacks
      - DOCKGE_STACKS_DIR=/opt/stacks
```


_Check config:_
```
docker compose config
```



_Start the Server:_
```
docker compose up -d
```



### Access Dockge Web UI:

Open your browser and go to: `http://<your-server-ip>:5001`
