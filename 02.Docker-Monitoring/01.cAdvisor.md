

## Docker cAdvisor: 
cAdvisor (Container Advisor) is a tool that helps monitor resource usage and performance characteristics of running containers, specifically for Docker and Kubernetes environments. It provides real-time metrics for CPU, memory, file, and network usage, making it an excellent choice for gaining insights into container performance.


### 1. cAdvisor on `docker run`: 

- DEPRECATED: is `google/cadvisor` and new images will NOT be pushed. Please use `gcr.io/cadvisor/cadvisor` instead.

```
docker run \
  --name=cadvisor \
  --publish=8080:8080 \
  --detach=true \
  --volume=/:/rootfs:ro \
  --volume=/var/run:/var/run:ro \
  --volume=/sys:/sys:ro \
  --volume=/var/lib/docker/:/var/lib/docker:ro \
  --volume=/dev/disk/:/dev/disk:ro \
  --privileged \
  --device=/dev/kmsg \
  gcr.io/cadvisor/cadvisor

```





### Web Console:
After running the command, you can access the cAdvisor web UI by visiting: `http://<host-ip>:8080` or `http://192.168.10.190:8080/docker/`




### 2. cAdvisor on `docker-compose.yaml`


```
services:
  cadvisor:
    image: gcr.io/cadvisor/cadvisor
    container_name: cadvisor
    privileged: true

    ports:
      - "8080:8080"
    volumes:

      - /:/rootfs:ro
      - /var/run:/var/run:ro
      - /sys:/sys:ro
      - /var/lib/docker/:/var/lib/docker:ro
      - /dev/disk/:/dev/disk:ro

    devices:
      - /dev/kmsg

    restart: always

```


```
docker-compose config
```


```
docker-compose up
```



### Web Console:

After running the command, you can access the cAdvisor web UI by visiting: `http://<host-ip>:8080` or `http://192.168.10.190:8080/docker/`
